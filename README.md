# Snake Redux

> **Work in Progress** - This project is currently under active development.

A fan-made game inspired by Snakes (2005), the classic 3D snake game originally released on the Nokia N-Gage platform. Built with modern web technologies.

## Features

- Dual-sided 3D maps (front and back traversal)
- Multiple grid types (square and hexagonal)
- Various obstacles, pickups, and power-ups
- Single-file offline deployment

## Tech Stack

- Three.js
- TypeScript
- Vite

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production (single HTML file)
npm run build
```
